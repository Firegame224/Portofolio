About Me

Saya Fiky Alrasya, lahir pada 24 April 2009 di Ajibarang, Kabupaten Banyumas, Jawa Tengah. Saya memiliki ketertarikan besar di dunia teknologi, terutama dalam pengembangan web dan aplikasi. Saat ini, saya merupakan siswa di SMK Telkom Purwokerto yg ingin menjadi Frontend Web Developer. Saya memiliki pengetahuan dalam Next.js, React, Tailwind CSS, dan JavaScript. Saya juga sedang mengembangkan Quatronime, sebuah website animelist yang menyajikan informasi lengkap tentang anime dan karakter. Selain coding, saya juga suka bermain game dan menonton anime, yang sering menjadi inspirasi dalam proyek-proyek saya. Saya percaya bahwa teknologi bisa menjadi solusi untuk banyak masalah, dan saya selalu berusaha untuk terus belajar serta berkembang di bidang ini.
